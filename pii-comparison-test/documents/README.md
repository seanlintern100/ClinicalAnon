# Test Documents

Place your clinical notes here as `.txt` files.

The comparison system will process all `.txt` files in this folder.

## Example Structure

```
documents/
├── clinical_note_1.txt
├── referral_letter.txt
├── discharge_summary.txt
└── README.md (this file)
```

## Tips

- Use real (or realistic) clinical notes for best results
- Include a variety of entity types: names, dates, NHI numbers, phone numbers
- Include edge cases that currently cause over-detection
